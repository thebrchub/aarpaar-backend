GET My Profile

/api/v1/users/me — Get the authenticated user's profile.

GET Search Users

/api/v1/users/search?query= — Search users by name or username.

PATCH Update Profile (Partial)

/api/v1/users/me — Update one or both fields. Only provided fields are changed.

(leave unchecked = no change)

PUT Replace Profile (Full)

/api/v1/users/me — Replace both username and name. Both fields required.

GET My Rooms

/api/v1/rooms — List all active chat rooms (excludes pending DM requests).

POST Create DM Room

/api/v1/rooms — Create a DM room. If target is private & not a friend, creates a pending DM request.

GET DM Requests (Inbox)

/api/v1/rooms/requests — Rooms where you have a pending DM invite (like Instagram message requests).

POST Accept / Reject DM Request

/api/v1/rooms/{roomId}/accept or /reject

GET Room Messages

/api/v1/rooms/{roomId}/messages — Fetch paginated messages in a room.

GET My Friends

/api/v1/friends — List all your friends.

GET Friend Requests

/api/v1/friends/requests?type=received|sent

POST Send Friend Request

/api/v1/friends/request — Send a friend request. Auto-accepts if mutual.

POST Accept / Reject / Remove

Accept or reject a pending request, or remove an existing friend.

POST Enter Match Queue

/api/v1/match/enter — Enter the matchmaking queue. Matches randomly with any available user.

POST Leave Match Queue

/api/v1/match/leave — Leave the matchmaking queue.

POST Match Action

/api/v1/match/action — Skip or block a stranger.

POST Report User

/api/v1/match/report — Report a user for moderation.
